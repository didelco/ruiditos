#Ruiditos

Ruiditos is another clone of [Noisli](http://noisli.com) now for raspberry pi, with phisical and mobile interface.

## Code:

### Nodejs

* Express
* Socket.io

### Html5

* Jquery
* JqueryUI
* Howler
